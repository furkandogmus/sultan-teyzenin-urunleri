# Sultan Teyze'nin Ürünleri

Ev yapımı börek, mantı, erişte, sarma ve diğer ürünleri sergileyen; müşterileri WhatsApp sipariş hattına yönlendiren tek sayfalık web sitesi.

## Düzenleme

WhatsApp numarası `script.js` dosyasındaki `WHATSAPP_NUMBER` alanından değiştirilir. Numara, ülke koduyla ve başında `+` olmadan yazılmalıdır. Arama bağlantıları HTML içindeki `tel:` alanlarında tutulur.

Site statik olduğu için doğrudan Netlify üzerinde yayınlanabilir.
