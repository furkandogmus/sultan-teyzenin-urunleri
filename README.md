# Sultan Teyze'nin Ürünleri

Ev yapımı hamur işi ürünlerini sergileyen ve müşterileri WhatsApp sipariş hattına yönlendiren tek sayfalık web sitesi.

## Düzenleme

WhatsApp numarası `script.js` dosyasındaki `WHATSAPP_NUMBER` alanından değiştirilir. Numara, ülke koduyla ve başında `+` olmadan yazılmalıdır.

Site statik olduğu için doğrudan Netlify üzerinde yayınlanabilir.
